# rishis12.github.io

Personal site + project demos. Jekyll, served on GitHub Pages.
In Progress

## Structure
- `_config.yml` — site config and nav
- `_layouts/` — shared page templates
- `_includes/` — nav and partials
- `_projects/` — project writeups (Jekyll collection)
- `assets/` — CSS and JS
- `fith/`, `donna/`, `btcbilboard/` — standalone interactive demos

## Local preview
    bundle install
    bundle exec jekyll serve
Then open http://localhost:4000
